# Future Interns – Data Science Task 2

## Project: Customer Churn & Retention Analysis

Tool Used: Power BI  
Dataset: Telco Customer Churn (Kaggle)

### Objectives:
- Analyze customer churn patterns
- Identify retention trends
- Find factors affecting churn
- Provide business recommendations

### Key Insights:
- Customers with short tenure churn more
- Month-to-month contracts have highest churn
- Customers without tech support churn more
- Long-term contracts improve retention
